# Phase 32-01: Content Optimization — SEO Content Enhancement

## Context

**Phase Goal**: Optimize page titles, descriptions, heading hierarchy, and content blocks for SEO.

**Current State**: 5 SEO page types exist (State, City, Neighborhood, Provider, Category) with basic metadata and single H1→H2 hierarchy. Missing: keyword-rich titles, FAQ sections, trust signals, semantic subheadings.

**Scope**: Create reusable content components and enhanced metadata helpers for all SEO pages.

## Tasks

### Task 1: Enhanced Metadata Helpers
**Type**: auto

Create extended metadata generators with keyword-rich titles and optimized descriptions.

**Files**:
- `src/lib/seo/metadata.ts` — Add `generateCategoryMetadata`, `generateProviderMetadata`, `generateNeighborhoodMetadata` functions with keyword-optimized patterns

**Changes**:
1. Add `generateNeighborhoodMetadata(neighborhood, city, state, stats)` with format: `"{treatment} Deals in {neighborhood}, {city} | Compare {n} Offers"`
2. Add `generateProviderMetadata(provider, city, state, stats)` with format: `"{provider} - {n} Medspa Deals in {city}, {state}"`
3. Add `generateCategoryMetadata(category, stats)` with format: `"{category} Treatments: {n} Deals from ${price} | CostFinders"`
4. Enhance descriptions with action verbs: "Compare", "Save up to", "Book today"

**Validation**: TypeScript compiles, functions return valid Metadata objects
